## Sim CLI commands

For your highly regulated chaos simulation:

- `help` - shows available commands
- `ls /<container>/<path>` - list files/folders
- `download /landing-egress/<path>` - download a file
- `upload /landing-ingress/<path>` - upload a file
- `roles` - list your current powers
- `rm -rf*` - totally safe cleanup operation

`curated` exists, but touching it triggers instant compliance theater.
